## 2. Sprint Goals

1. Complete ADC pipeline implementation (carry-over from SP-07)
2. Achieve PFC open-loop PWM at 65 kHz with correct dead-time and interleaving
3. Implement SRF-PLL and verify grid lock at all input voltage corners
4. Begin LLC open-loop frequency sweep

## 4. Risks and Blockers

- **PFC shoot-through** — Dead-time must be verified before applying power; scope monitoring essential
- **PLL lock failure** — At voltage extremes (260 V, 530 V), PLL may not lock within 2 s timeout
- **LLC resonant frequency mismatch** — Actual fr may differ from design; affects gain curve
