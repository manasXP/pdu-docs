# 02 – Power-On Sequence and Ramp Control

> [!note] Scope
> This document covers **firmware-side sequencing**: peripheral initialization, PLL lock detection, PFC/LLC soft-start ramp profiles, and shutdown sequencing. For **hardware-side inrush management** (NTC pre-charge, bypass relay, contactor sizing, cold-start analysis), see [[08-Power-On Sequence and Inrush Management]]. For HRTIM and ADC resource allocation, see [[06-Firmware Architecture]].

## 1. Initialization Sequence (POWER_ON State)

After reset, the firmware initializes peripherals in a deterministic order. Any failure transitions to `APP_DISABLED` (see [[01-Application State Machine]]).

| Step | Peripheral | Action | Duration | Failure Mode |
|------|-----------|--------|----------|--------------|
| 1 | System clock | Configure PLL: HSE → 170 MHz | < 1 ms | Clock fail → NMI |
| 2 | GPIO | Configure all pins (outputs low, fault inputs) | < 1 ms | — |
| 3 | HRTIM DLL | `HAL_HRTIM_DLLCalibrationStart()`, poll DLLRDY | 14 ms typ | Timeout 50 ms → DISABLED |
| 4 | ADC1 self-cal | `LL_ADC_StartCalibration()` single-ended + differential | 2 ms each | Retry ×3, then DISABLED |
| 5 | ADC2 self-cal | Same | 2 ms | Same |
| 6 | ADC3 self-cal | Same | 2 ms | Same |
| 7 | ADC4 self-cal | Same | 2 ms | Same |
| 8 | ADC5 self-cal | Same | 2 ms | Same |
| 9 | DMA | Configure channels for regular ADC groups | < 1 ms | — |
| 10 | HRTIM timers | Configure A–F, master, fault inputs, preload enable | < 1 ms | — |
| 11 | FDCAN1 | Init 500 kbps, configure Rx FIFO, read node ID DIP | < 1 ms | — |
| 12 | CORDIC/FMAC | Load Park transform coefficients, IIR filter coefficients | < 1 ms | — |
| 13 | TIM6 | 1 kHz tick for main loop scheduling | < 1 ms | — |
| 14 | Flash | Read fault log pointer, calibration data | 1 ms | Use defaults if corrupt |
| 15 | Self-test | Read V_aux_12V, verify in range (11.4–12.6 V) | 5 ms | DISABLED if out of range |

**Total init time: ~30 ms** (dominated by HRTIM DLL + 5× ADC self-cal)

```c
bool System_Init(void)
{
    /* Steps 1-2: Clock and GPIO (generated by CubeMX) */
    SystemClock_Config();
    MX_GPIO_Init();

    /* Step 3: HRTIM DLL calibration */
    HAL_HRTIM_DLLCalibrationStart(&hhrtim1, HRTIM_CALIBRATIONRATE_3);
    uint32_t t0 = HAL_GetTick();
    while (!__HAL_HRTIM_GET_FLAG(&hhrtim1, HRTIM_FLAG_DLLRDY)) {
        if ((HAL_GetTick() - t0) > 50) return false;
    }

    /* Steps 4-8: ADC self-calibration (all 5 instances) */
    ADC_TypeDef *adcs[] = {ADC1, ADC2, ADC3, ADC4, ADC5};
    for (int i = 0; i < 5; i++) {
        LL_ADC_StartCalibration(adcs[i], LL_ADC_SINGLE_ENDED);
        while (LL_ADC_IsCalibrationOnGoing(adcs[i])) {}
        LL_ADC_StartCalibration(adcs[i], LL_ADC_DIFFERENTIAL_ENDED);
        while (LL_ADC_IsCalibrationOnGoing(adcs[i])) {}
    }

    /* Steps 9-14: DMA, HRTIM timers, CAN, co-processors, flash */
    ADC_DMA_Init();         /* see [[06-ADC Pipeline and DMA Configuration]] */
    HRTIM_AllTimers_Init();
    FDCAN1_Init();
    CORDIC_FMAC_Init();
    TIM6_Init_1kHz();
    FaultLog_Init();

    /* Step 15: Auxiliary rail check */
    return AuxRail_OK();
}
```

## 2. SRF-PLL Pre-Lock (PLL_LOCK State)

The PLL must lock to the grid angle `theta` before enabling PFC current control. Running the current loop with an incorrect angle produces large transient currents.

### 2.1 Lock Sequence

1. **Enable ADC injected channels** for grid voltages (V_A, V_B, V_C) and bus voltage
2. **Start SRF-PLL** with omega-hat initialized to 0 (not nominal — prevents lock to wrong harmonic)
3. **Ramp omega-hat** from 0 toward expected grid frequency using a rate-limited integrator:
   - Rate limit: 500 rad/s² (reaches 314 rad/s in ~630 ms for 50 Hz grid)
   - This prevents the PLL from locking to a harmonic or aliased frequency
4. **Monitor V_q convergence**: PLL is locked when |V_q| < V_q_threshold for N consecutive samples

### 2.2 Lock Detection

```c
#define PLL_VQ_THRESHOLD    5.0f     /* Volts — V_q close to zero */
#define PLL_LOCK_COUNT      20       /* 20 consecutive samples at 48-65 kHz */
#define PLL_LOCK_TIMEOUT_MS 2000     /* 2 second timeout */

typedef struct {
    float omega_hat;        /* estimated grid angular frequency (rad/s) */
    float theta_hat;        /* estimated grid angle (rad) */
    float vq_filtered;      /* filtered V_q for lock detection */
    uint16_t lock_counter;  /* consecutive samples within threshold */
    bool locked;
} PLL_State_t;

bool PLL_CheckLock(PLL_State_t *pll, float vq_meas)
{
    /* EMA filter on |V_q| for noise rejection */
    pll->vq_filtered = 0.95f * pll->vq_filtered + 0.05f * fabsf(vq_meas);

    if (pll->vq_filtered < PLL_VQ_THRESHOLD) {
        pll->lock_counter++;
        if (pll->lock_counter >= PLL_LOCK_COUNT) {
            pll->locked = true;
            return true;
        }
    } else {
        pll->lock_counter = 0;
    }
    return false;
}
```

### 2.3 Omega-Hat Ramp

```c
#define OMEGA_RAMP_RATE   500.0f   /* rad/s^2 */
#define OMEGA_50HZ        314.16f  /* 2*pi*50 */
#define OMEGA_60HZ        376.99f  /* 2*pi*60 */

void PLL_OmegaRamp(PLL_State_t *pll, float dt)
{
    /* Before lock, ramp omega_hat toward expected range */
    if (!pll->locked) {
        float omega_target = OMEGA_50HZ;  /* or OMEGA_60HZ per config */
        if (pll->omega_hat < omega_target) {
            pll->omega_hat += OMEGA_RAMP_RATE * dt;
            if (pll->omega_hat > omega_target)
                pll->omega_hat = omega_target;
        }
    }
    /* After lock, PLL PI controller drives omega_hat directly */
}
```

## 3. PFC Soft-Start (SOFT_START_PFC State)

### 3.1 Ramp Profile

The PFC soft-start ramps the d-axis current reference `I_d*` linearly from 0 to rated value over 200 ms while the bus voltage rises to target.

```
I_d* (A)
  |
  |rated ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ /────────
  |                                                /
  |                                             /
  |                                          /
  |                                       /
  |                                    /
  |                                 /
  |──────────────────────────────/
  0                              |              |
  t=0                          +200 ms        +250 ms (V_bus stable)
```

### 3.2 Anti-Windup Pre-Load

Before enabling the current loop, pre-load the voltage PI integrator to prevent a large transient:

```c
/* Pre-load: set integrator output to approximately the duty cycle
 * needed to maintain current bus voltage (from pre-charge) */
float vbus_initial = ADC_Read_VbusFilt();
float d_preload = vbus_initial / V_GRID_PEAK;  /* approximate duty */
Vbus_PI.integrator = d_preload * Vbus_PI.Ki_inv;
```

### 3.3 Bypass Relay Engagement

The NTC bypass relay closes once V_bus > 80% of target, removing the NTC from the inrush path. This is coordinated with the FW ramp:

```
Event timeline:
  t=0:       PFC HRTIM outputs enabled, I_d* = 0
  t+10ms:    I_d* begins linear ramp (rate = I_rated / 200 ms)
  t+100ms:   V_bus reaches ~80% target → close NTC bypass relay
  t+200ms:   I_d* = I_rated; V_bus approaching target
  t+250ms:   V_bus stable within ±10 V of target for 50 ms → ready for LLC
```

### 3.4 PFC Soft-Start Pseudocode

```c
typedef struct {
    float id_ref;          /* current d-axis reference (A) */
    float id_target;       /* final I_d* at full load */
    float ramp_rate;       /* A/ms */
    bool  bypass_closed;   /* NTC bypass relay state */
    bool  complete;
} PFC_SoftStart_t;

void PFC_SoftStart_Begin(void)
{
    g_pfc_ss.id_ref = 0.0f;
    g_pfc_ss.id_target = I_RATED_D;
    g_pfc_ss.ramp_rate = I_RATED_D / 200.0f;  /* A/ms */
    g_pfc_ss.bypass_closed = false;
    g_pfc_ss.complete = false;

    /* Pre-load voltage PI integrator */
    float vbus = ADC_Read_VbusFilt();
    Vbus_PI_Preload(vbus);

    /* Enable HRTIM PFC outputs (TA, TB, TC) */
    HRTIM_EnableOutputs(HRTIM_OUTPUT_TA1 | HRTIM_OUTPUT_TA2 |
                        HRTIM_OUTPUT_TB1 | HRTIM_OUTPUT_TB2 |
                        HRTIM_OUTPUT_TC1 | HRTIM_OUTPUT_TC2);
}

void PFC_SoftStart_Tick(void)  /* called at 1 kHz from main loop */
{
    if (g_pfc_ss.complete) return;

    /* Linear ramp of I_d* */
    if (g_pfc_ss.id_ref < g_pfc_ss.id_target) {
        g_pfc_ss.id_ref += g_pfc_ss.ramp_rate;
        if (g_pfc_ss.id_ref > g_pfc_ss.id_target)
            g_pfc_ss.id_ref = g_pfc_ss.id_target;
    }

    /* NTC bypass relay engagement */
    float vbus = ADC_Read_VbusFilt();
    if (!g_pfc_ss.bypass_closed && vbus > 0.80f * VBUS_TARGET) {
        GPIO_Set(NTC_BYPASS_RELAY, 1);
        g_pfc_ss.bypass_closed = true;
    }

    /* Update current reference for ISR */
    __atomic_store_float(&g_id_ref, g_pfc_ss.id_ref);
}
```

## 4. LLC Soft-Start (SOFT_START_LLC State)

### 4.1 Frequency-Down Ramp

The LLC starts at maximum frequency (minimum gain) and ramps down toward resonance. This prevents output voltage overshoot at startup.

```
f_sw (kHz)
  300 |────┐
      |    │
      |    │
      |     \
      |      \
      |       \
      |        \
  150 |─ ─ ─ ─ ─\─ ─ ─ ─ ─ ── f_resonant
      |           \────────────── f_operating (regulated by CV loop)
      |
      0    100   200   300   400  500 ms
           └─phase 1 enabled
                └─phase 2 enabled (+100 ms)
                     └─phase 3 enabled (+200 ms)
                          └─contactor close
                               └─CV loop takes over
```

### 4.2 Staggered Phase Enable

The three LLC phases are enabled sequentially with 100 ms spacing to limit inrush to the output capacitors:

| Time | Action |
|------|--------|
| t+0 ms | Enable HRTIM Timer D (LLC Phase 1) at f_max = 300 kHz |
| t+100 ms | Enable HRTIM Timer E (LLC Phase 2) at current f_sw |
| t+200 ms | Enable HRTIM Timer F (LLC Phase 3) at current f_sw |
| t+300 ms | Close output contactor (V_out within ±5% of target) |
| t+400 ms | Transfer control to CV/CC outer loop |

### 4.3 LLC Soft-Start Pseudocode

```c
typedef struct {
    float    f_sw;               /* current switching frequency (Hz) */
    float    f_target;           /* target frequency from CV loop */
    float    ramp_rate;          /* Hz/ms (negative: ramping down) */
    uint8_t  phases_enabled;     /* bitmask: bit0=D, bit1=E, bit2=F */
    uint32_t phase_enable_time;  /* timestamp of last phase enable */
    bool     contactor_closed;
    bool     complete;
} LLC_SoftStart_t;

void LLC_SoftStart_Begin(void)
{
    g_llc_ss.f_sw = F_SW_MAX;              /* 300 kHz */
    g_llc_ss.ramp_rate = -400.0f;          /* -400 Hz/ms → 300→150 kHz in 375 ms */
    g_llc_ss.phases_enabled = 0;
    g_llc_ss.contactor_closed = false;
    g_llc_ss.complete = false;

    /* Enable first LLC phase */
    HRTIM_SetPeriod_Hz(HRTIM_TIMER_D, g_llc_ss.f_sw);
    HRTIM_EnableOutputs(HRTIM_OUTPUT_TD1 | HRTIM_OUTPUT_TD2);
    g_llc_ss.phases_enabled = 0x01;
    g_llc_ss.phase_enable_time = HAL_GetTick();
}

void LLC_SoftStart_Tick(void)  /* called at 1 kHz from main loop */
{
    if (g_llc_ss.complete) return;

    uint32_t now = HAL_GetTick();
    uint32_t dt_phase = now - g_llc_ss.phase_enable_time;

    /* Staggered phase enable (100 ms spacing) */
    if (g_llc_ss.phases_enabled == 0x01 && dt_phase >= 100) {
        HRTIM_SetPeriod_Hz(HRTIM_TIMER_E, g_llc_ss.f_sw);
        HRTIM_EnableOutputs(HRTIM_OUTPUT_TE1 | HRTIM_OUTPUT_TE2);
        g_llc_ss.phases_enabled = 0x03;
        g_llc_ss.phase_enable_time = now;
    }
    if (g_llc_ss.phases_enabled == 0x03 && dt_phase >= 100) {
        HRTIM_SetPeriod_Hz(HRTIM_TIMER_F, g_llc_ss.f_sw);
        HRTIM_EnableOutputs(HRTIM_OUTPUT_TF1 | HRTIM_OUTPUT_TF2);
        g_llc_ss.phases_enabled = 0x07;
        g_llc_ss.phase_enable_time = now;
    }

    /* Frequency ramp (only while all 3 phases are ramping) */
    if (g_llc_ss.phases_enabled == 0x07) {
        g_llc_ss.f_sw += g_llc_ss.ramp_rate;  /* ramp_rate is negative */
        if (g_llc_ss.f_sw < F_SW_MIN) g_llc_ss.f_sw = F_SW_MIN;

        /* Update all three phase periods atomically */
        HRTIM_SetPeriod_Hz_AllLLC(g_llc_ss.f_sw);  /* writes Master + D/E/F */
    }

    /* Contactor close once V_out is within range */
    float vout = ADC_Read_VoutFilt();
    if (!g_llc_ss.contactor_closed && g_llc_ss.phases_enabled == 0x07
        && fabsf(vout - g_vout_target) < 0.05f * g_vout_target) {
        Contactor_Close();
        g_llc_ss.contactor_closed = true;
    }

    /* Complete: hand off to CV/CC loop */
    if (g_llc_ss.contactor_closed && fabsf(vout - g_vout_target) < 0.02f * g_vout_target) {
        LLC_CV_Loop_Enable();
        g_llc_ss.complete = true;
    }
}
```

## 6. Shutdown Sequence (SHUTDOWN State)

Shutdown is the reverse of startup. The goal is zero-current turn-off of all switches before opening contactors.

| Step | Action | Duration |
|------|--------|----------|
| 1 | Ramp I_out reference to 0 (LLC CC mode, rate = −100 A/s) | 100–1000 ms |
| 2 | Disable LLC HRTIM outputs (F, E, D — reverse order) | immediate |
| 3 | Ramp I_d* to 0 (PFC current reference) | 200 ms |
| 4 | Disable PFC HRTIM outputs (C, B, A) | immediate |
| 5 | Open output contactor | 10 ms relay delay |
| 6 | Open NTC bypass relay (reconnect NTC for next startup) | 10 ms |
| 7 | Transition to STANDBY | — |

**Total shutdown: ~500 ms nominal, 3 s timeout.**

```c
void Shutdown_Tick(void)
{
    switch (g_shutdown_step) {
    case 0:  /* Ramp LLC output to zero */
        g_iout_ref -= 100.0f * 0.001f;  /* -100 A/s at 1 kHz tick */
        if (g_iout_ref <= 0.5f) {
            g_iout_ref = 0.0f;
            HRTIM_DisableOutputs(HRTIM_OUTPUT_TF1 | HRTIM_OUTPUT_TF2);
            HRTIM_DisableOutputs(HRTIM_OUTPUT_TE1 | HRTIM_OUTPUT_TE2);
            HRTIM_DisableOutputs(HRTIM_OUTPUT_TD1 | HRTIM_OUTPUT_TD2);
            g_shutdown_step = 1;
            g_shutdown_step_time = HAL_GetTick();
        }
        break;

    case 1:  /* Ramp PFC I_d* to zero */
        g_id_ref -= I_RATED_D / 200.0f;  /* same rate as soft-start */
        if (g_id_ref <= 0.1f) {
            g_id_ref = 0.0f;
            HRTIM_DisableOutputs(HRTIM_OUTPUT_TC1 | HRTIM_OUTPUT_TC2);
            HRTIM_DisableOutputs(HRTIM_OUTPUT_TB1 | HRTIM_OUTPUT_TB2);
            HRTIM_DisableOutputs(HRTIM_OUTPUT_TA1 | HRTIM_OUTPUT_TA2);
            g_shutdown_step = 2;
        }
        break;

    case 2:  /* Open contactors and relays */
        Contactor_Open();
        GPIO_Set(NTC_BYPASS_RELAY, 0);
        g_shutdown_step = 3;
        break;

    case 3:  /* Done */
        g_shutdown_complete = true;
        break;
    }
}
```
